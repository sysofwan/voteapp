About VoteApp

Conduct polls in real time.

Polls are created and hosted through unique sessions. Each session tracks data of unique voters, poll results and time.

Vote using any device.

VoteApp is a web application. Any device with internet access can use the app. With Twilio, voters can also send text messages to vote.

Accessible to all.

We believe that voting should be available for everyone. Our app is easy to use and is free under the open source initiative.
